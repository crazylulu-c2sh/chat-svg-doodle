아래의 규칙으로 **고양이**를 그려줘.

## 필수

**HTML/CSS/SVG 사용**
- <path> 위주
- stroke-opacity를 서로 다르게
- stroke-width를 약간씩 다르게

- 크레파스로 문지르듯 그린 선
- 한 선을 2~3번 겹쳐 긋되 완전히 같은 경로는 아님
- 선의 두께와 진하기가 중간중간 끊기듯 변함
- 종이 결 때문에 가장자리가 거칠게 깨짐
- 일정하지 않은 압력 (약해졌다 강해졌다)
- 흰 배경
- 선 색상 7개 이하로 제한
- 선 개수 1초에 3개 이하로 제한
- 한 번에 그린 선, 되돌아가지 않음
- 비율이 틀려도 수정하지 말 것
- 어린이가 기억으로 그린 그림
- 대칭 금지
- 마우스로 그린 느낌 (직선 + 떨림)
- 망친 흔적을 의도적으로 만들라

## 애니메이션
- **5초** 안에 그린 낙서
- 그려지는 애니메이션
- 한번에 선은 하나만 애니메이션


## 금지
- 균일한 stroke
- 매끄러운 곡선
- 한 번에 완벽한 선
- 디지털 느낌
